# 🏢 Coworking Space System (Java OOP)

## 📝 Project Description
A console‑based coworking space reservation system developed in Java using Object‑Oriented Programming principles. The system allows different user roles (Admin, User, Instructor) to manage and book workspaces.

## 🛠️ Technologies Used
- Java (OOP)
- File I/O (text files for persistence)
- Git & GitHub

## ✨ Key Features
- **Multi‑role access**: Admin, User, Instructor.
- **Manage slots and rooms** (Admin).
- **Reserve workspaces** (User/Instructor).
- **Data stored persistently** in `.txt` files.
- **Modular design** with separate classes for each entity.
